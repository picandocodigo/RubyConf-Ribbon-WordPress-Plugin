Add a RubyConf Uruguay ribbon to your blog.

== Description ==

A simple WordPress plugin to add a RubyConf Uruguay ribbon to your blog. Visit http://rubyconfuruguay.org/ to know more about Ruby Conf Uruguay.

== Installation ==

1. Upload the `rubyconfuy-ribbon` directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. The ribbon should now show up in your website :)
